## Sobre mim

Desenvolvedor Full Stack com **3+ anos de experiência** em aplicações web, automações, integrações e sistemas corporativos. Hoje atuo na **Schwarz S/A**, com foco em **.NET, Blazor, Python, SQL Server, GitHub Actions, Docker, Azure DevOps** e ecossistema **TOTVS (Fluig/Datasul)**.

Além dos projetos próprios, também contribuo com **open source no ecossistema MudBlazor**, levando correções reais para componentes de interface usados em produção.
